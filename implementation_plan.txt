# Email System Comprehensive Fix - Implementation Plan

## Overview
Fix email sending across all CRM components in 7 phases, organized by component.

---

## Phase 1: SendQuoteModal.tsx (Critical - Blocks Quote Emails)
**Affects:** QuoteDetailModal.tsx, QuoteManagement.tsx

### Issues
- [ ] Not wired to `send-quote-email` edge function
- [ ] Hardcoded "Momentum CRM" → should use company name
- [ ] Quote acceptance link not included

1. Wire `supabase.functions.invoke('send-quote-email')` 
2. Fetch organisation name dynamically
3. Pass `include_acceptance_link: true`
